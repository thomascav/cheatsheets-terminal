Terminal - CheatSheets -

Quelques commandes du terminal qui dépannent...

  • ls
  • cd
  • cp
  • mv
  • rm
  • mkdir
  • man
La commande ls permet de lister les dossiers et les fichiers d'un répertoire.

ls -a permet le lister tous les éléments d'un répertoire, même les cachés

ls -l liste tous les éléments sous forme de tableau de manière détaillée.

La commande cd permet de changer de répertoire.

cd .. permet de remonter au dossier parent.

cd / permet de remonter à la racine du disque.

pwd affiche le chemin absolu du répertoire courant.

cd /x/xx aller directement dans le sous dossier xx du dossier x.

La commande cp permet de copier un fichier, le 1er dans le suivant, peut servir à renommer.
La commande mv correspond à la fonction "couper".

mv X/x.txt y/ déplace le fichier x.txt dans le dossier y

mv x.txt y.txt renomme le fichier x.txt en y.txt

La commande mkdir permet de créer un dossier.

mkdir x permet de créer le dossier x

mkdir -p x/y/z permet de créer l'arborescence x/y/z

La commande man permet d'afficher toutes les options de la commande (tapée après man).
